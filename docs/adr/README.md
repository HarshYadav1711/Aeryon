# Architecture Decision Records

This directory contains Architecture Decision Records (ADRs) for Aeryon.

## Format

Each ADR follows this structure:

```
# NNNN. Title

Date: YYYY-MM-DD
Status: proposed | accepted | deprecated | superseded

## Context

## Decision

## Consequences
```
