# Aeryon Frontend

Local-development dashboard for the deterministic synthetic sensor vertical slice.

## Setup

```bash
cp .env.example .env
npm install
```

## Two-process workflow

Terminal 1 (repository root):

```bash
cargo run --bin server
```

Terminal 2:

```bash
npm run dev
```

Open the Vite URL (default `http://127.0.0.1:5173`).

Environment:

- `VITE_AERYON_API_URL` — REST base URL (default `http://127.0.0.1:8080`)
- `VITE_AERYON_WS_URL` — WebSocket base URL (default `ws://127.0.0.1:8080`)

## Scripts

| Command | Purpose |
|---------|---------|
| `npm run dev` | Vite development server |
| `npm run build` | Production build |
| `npm run test` | Vitest unit tests |
| `npm run lint` | Oxlint |

## Notes

- Data is generated by the deterministic synthetic source on the Rust server.
- This UI validates platform integration. WiFi CSI ingestion is not implemented.
- The HTTP API is local-development infrastructure at this stage.
